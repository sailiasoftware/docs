This page gives you a quick reference for the tasks you perform most often in Sailia. Each entry links to the full guide for more detail.

## Bookings

| Task | How to do it |
| --- | --- |
| **Create a walk-in booking** | Open the [POS](/point-of-sale/overview), add the activity to the basket, collect attendee details, and take payment |
| **View today's bookings** | Open the [bookings dashboard](/activities/managing-bookings) and filter by today's date |
| **Cancel a booking** | Open the booking, select **Cancel**, choose a [refund method](/activities/cancellations-and-refunds#refund-methods), and confirm |
| **Move a booking to a different session** | Open the booking and [change the session](/activities/managing-bookings#edit-a-booking) to a different date or time |
| **Resend a confirmation email** | Open the booking details and select [resend confirmation](/activities/managing-bookings#resend-confirmation-emails) |
| **Check who is attending a session** | Open the session in your [schedule view](/activities/managing-bookings#schedule-filters) to see the attendee list |

## Payments

| Task | How to do it |
| --- | --- |
| **Process a card refund** | Open the booking, select **Refund**, choose **Original payment method**, and confirm — see [refunds](/payments/stripe#refunds) |
| **Issue store credit** | Open the booking, select **Refund**, enable **Issue as credit** — a [gift card](/pricing/gift-cards) is created automatically |
| **Take a cash payment** | [Open a till shift](/point-of-sale/overview#open-a-shift), then select **Cash** as the payment method at checkout |
| **Send a payment link** | In the POS, select **Send payment link** and enter the customer's email — see [card readers](/point-of-sale/card-readers#send-a-payment-link-by-email) |
| **View today's revenue** | Open the [financial dashboard](/payments/reporting) and filter by today's date |

## Customers

| Task | How to do it |
| --- | --- |
| **Find a customer** | Search by name or email in the [customer dashboard](/customers/overview) |
| **Check waiver status** | Open the booking or customer profile to see [waiver completion](/customers/waivers#viewing-completed-waivers) |
| **Add a note to a customer** | Open their profile, go to the [Notes tab](/customers/overview#notes), and save your note |
| **Send a waiver manually** | Go to the [waivers dashboard](/customers/waivers#send-a-waiver-to-customers), select the waiver and recipients, set a due date, and send |

## Staff

| Task | How to do it |
| --- | --- |
| **Add a new staff member** | Go to [staff management](/staff/management) and select **Create Staff Member** |
| **Approve a time-off request** | Open the request in the [staff dashboard](/staff/availability#reviewing-requests) and select **Approve** |
| **Review a timesheet invoice** | Open the [submitted invoice](/staff/timesheet-invoices#reviewing-invoices) and approve, reject, or void it |

## Promotions

| Task | How to do it |
| --- | --- |
| **Create a discount code** | Go to [discount codes](/pricing/discount-codes#create-a-discount-code) and configure the code, type, amount, and any limits |
| **Sell a gift card** | Customers can purchase through your [booking page](/pricing/gift-cards#selling-gift-cards), or staff can sell through the [POS](/point-of-sale/overview) |
| **Set up a membership** | Go to [memberships](/pricing/memberships#create-a-membership-plan) and configure the plan, billing interval, and benefits |

## Integrations

| Task | How to do it |
| --- | --- |
| **Connect Xero** | Go to [accounting sync](/integrations/accounting-sync) settings, connect your Xero account, and configure accounting categories |
| **Set up an automated email** | Create a [workflow](/integrations/workflows) with a trigger, optional delay, and message template |
| **Enable booking tracking** | Connect [Google Tag Manager or PostHog](/integrations/booking-tracking) to capture booking funnel events |
| **List activities on Adventuro** | Activate the [Adventuro integration](/integrations/adventuro) and share your API key |

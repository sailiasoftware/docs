The Memberships dashboard is organised into four tabs. Each one gives you a different lens on your member base.

## Overview

Graphs and statistics about your member base over time. From here you can also export data — payments, member lists, and any custom exports configured for your account.

## Directory

A list of every member you have. A single subscription can include several people (family or team plans), and the directory lists each person individually. Clicking any name opens the parent subscription, where you can see all linked members.

Filter the directory using the search box or the built-in filters. For more granular slicing, run an export.

## Memberships

Your main memberships — one card per membership, with name, photos, description, and benefits. Pricing is set at the **plan** level, not on the membership itself.

From this tab you can:

- Create a new membership.
- Open an existing one to edit its benefits and plans.
- See the members associated with each plan.

## Add-ons

Every [add-on](/pricing/membership-addons) in your account, with the plans each one is linked to. You can see how many people have purchased each add-on and which specific plan they hold it on — useful for spotting strong-performing add-ons across multiple memberships.

For example, *Boat storage* might be attached to several plans across different memberships. The Add-ons tab shows total uptake and the breakdown by plan in one place.

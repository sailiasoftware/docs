Sailia runs entirely in your web browser — there is nothing to install. This page covers what you need to get the best experience.

## Supported browsers

Sailia works with the latest versions of all major browsers:

| Browser | Supported |
| --- | --- |
| **Google Chrome** | Latest 2 versions |
| **Mozilla Firefox** | Latest 2 versions |
| **Apple Safari** | Latest 2 versions |
| **Microsoft Edge** | Latest 2 versions |

<Warning>
  Internet Explorer is not supported. If you or your customers are using IE, switch to a modern browser for the best experience.
</Warning>

## Device support

### Admin dashboard

The Sailia admin dashboard (where you manage bookings, staff, and settings) works best on:

- **Desktop and laptop computers** — full feature set with the best experience
- **Tablets** — functional for most tasks including POS operations

<Tip>
  If you use the [point of sale](/point-of-sale/overview) on a tablet, pair it with a [Stripe card reader](/point-of-sale/card-readers) for seamless in-person payments.
</Tip>

### Customer booking page

Your [booking page](/getting-started/booking-page) is responsive and works across all device types:

- **Desktop** — full browsing and checkout experience
- **Tablet** — optimized layout for touch interaction
- **Mobile** — streamlined design for booking on the go

## Network requirements

Sailia requires an active internet connection for all operations. Key considerations:

| Feature | Network requirement |
| --- | --- |
| **Online bookings** | Standard internet connection for your customers |
| **Admin dashboard** | Stable connection for real-time booking updates |
| **Card reader payments** | Wi-Fi or ethernet connection to the [card reader](/point-of-sale/card-readers) and internet access for Stripe processing |
| **POS operations** | Continuous connection — POS cannot process payments offline |

<Note>
  Card reader payments require both the reader and your device to be connected to the internet. If your venue has unreliable connectivity, consider a wired ethernet connection for your card reader where possible.
</Note>

## Accessibility

Sailia aims to provide an accessible experience for all users. The booking page follows web accessibility best practices including:

- Keyboard navigation support
- Screen reader compatibility
- Sufficient color contrast ratios
- Clear form labels and error messages

If you encounter accessibility issues, contact [Sailia support](mailto:support@sailia.com) so the team can address them.

Images appear all over Sailia — your logo, activity photos, member profiles, and shop products. This page covers the formats and sizes that work everywhere, plus tips for getting the best results.

## Supported formats

Most image upload fields in Sailia accept:

- **JPEG** (`.jpg`, `.jpeg`) — best for photos
- **PNG** (`.png`) — best when you need a transparent background

Some upload fields (such as general attachments) also accept other formats like GIF or WebP. If the field is specifically asking for a photo or image, stick to JPEG or PNG.

## File size and dimensions

| Setting | Recommended value |
| --- | --- |
| **Maximum file size** | Up to 50 MB per image |
| **Aspect ratio for banners and activity photos** | 16:9 |
| **Aspect ratio for profile or square thumbnails** | 1:1 (square) |
| **Logo orientation** | Horizontal, with clear space around the mark |

Sailia automatically generates optimised versions of every image you upload, so always use the largest reasonable resolution within the size limit. Sharper source images look better on mobile and large screens alike.

## Tips by image type

- **Logos** — horizontal images with plenty of breathing room. Tall or very narrow logos appear small in the header.
- **Profile pictures** — square images work best, especially where Sailia crops them into circles or thumbnails.
- **Banners and hero images** — landscape orientation, wider than tall.
- **Activity photos** — 16:9 landscape, ideally showing what customers will actually do.

## Browsers and devices

Image uploads work on desktop and mobile, but rely on modern web features. Use a recent version of:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Apple Safari

If an upload stalls or fails:

1. Refresh the page and try again.
2. Temporarily disable any VPN.
3. Switch to a different browser.
4. For large files, use a strong Wi-Fi or wired connection.

## HEIC and HEIF files from iPhone

iPhones save photos in **HEIC** or **HEIF** by default, but most web uploaders need **JPEG** or **PNG**. If your upload is rejected:

- Export or duplicate the photo as JPEG before uploading, or
- Share it via AirDrop or email using the **Most Compatible** option, which converts it to JPEG automatically.

## File name tips

If you hit an unusual upload error, try renaming the file. Stick to:

- Letters and numbers
- Spaces, hyphens, or underscores

Avoid special characters or emojis in file names.

## Still stuck?

If an upload keeps failing, [contact support](/help/contact) with:

- The page you were on (for example, branding, session photos, profile picture)
- The file type and size
- The exact error message

Upload limits can vary slightly by area, so this detail helps support give you a precise answer.
